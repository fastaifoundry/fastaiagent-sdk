# Contributing to FastAIAgent SDK

FastAIAgent is **open source** under the [Apache License 2.0](LICENSE) — you are
free to **use, modify, fork, and redistribute** it under those terms.

The upstream project, however, is **closed to external code contributions**. It is
developed and maintained solely by **FastAIFoundry**, which keeps the SDK's
architecture, public APIs, and wire contract coherent with the FastAIAgent
platform. This is a deliberate "open source, closed contribution" model.

## What you can do

- **Use, modify, and fork** the SDK freely under Apache-2.0.
- **File issues** — bug reports, feature requests, integration requests, and
  documentation feedback are genuinely welcome and shape the roadmap:
  [GitHub Issues](https://github.com/fastaifoundry/fastaiagent-sdk/issues).
- **Maintain your own fork** with the changes you need (Apache-2.0 permits this).

## What we don't accept

- **External pull requests are not accepted or merged.** Please don't open a PR
  with code changes — it will be closed with a pointer to this policy. Open an
  **issue** instead to report the bug or propose the change, and we'll track it.

So: **file issues, not PRs.**

## Reporting issues

Use [GitHub Issues](https://github.com/fastaifoundry/fastaiagent-sdk/issues). A
clear reproduction, the `fastaiagent` version, and your environment help a lot.

## Code of Conduct

Participation in the issue tracker and other community spaces is governed by our
[Code of Conduct](CODE_OF_CONDUCT.md).
